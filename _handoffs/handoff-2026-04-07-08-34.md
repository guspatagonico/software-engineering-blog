## Decisions Made

- **Matrix background toggle**: Instead of unmounting the component (which would lose state), the animation loop checks visibility and skips rendering entirely when hidden. This is more efficient than dynamic import.
- **Touch devices**: Dodecahedron hover effect is disabled on touch devices so tapping doesn't stop rotation — click still works for toggle.
- **Visibility detection**: MatrixBackground monitors both the custom event `toggle-matrix-background` and body class changes via MutationObserver.
- **Dodecahedron z-index**: Set to 40 to sit below navbar (50/90/100) and footer (50), avoiding overlap with hamburger menu on mobile.
