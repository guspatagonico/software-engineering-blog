# Domain: project_guidelines

## Purpose
Capture the repository-wide build, lint, test, and run conventions that keep the interactive blog consistent across environments.

## Scope
Included in this domain:
- pnpm driver scripts
- testing workflows (Vitest + Playwright)
- lint/typecheck/format gates

Excluded from this domain:
- feature-specific component implementations
- blog narrative content
