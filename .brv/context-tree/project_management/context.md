# Domain: project_management

## Purpose
Store session handoff summaries that capture completed work, decisions, blockers, and pending items so future contributors can resume with context.

## Scope
Included in this domain:
- Completed tasks
- Decisions made
- Files touched
- Pending actions
- Blockers

Excluded from this domain:
- Source code implementation details outside of summary context
