## Narrative
### Highlights
Applies to HTML, MDX, and JSX files to maintain consistent tag structure.

### Rules
Rule: Always write HTML/MDX/JSX tags so the closing angle bracket '>' stays on the same line as the tag boundary (avoid leading '>' on following lines).

## Facts
- **html_tag_formatting**: Always write HTML/MDX/JSX tags so the closing angle bracket '>' stays on the same line as the tag boundary (avoid leading '>' on following lines). [preference]
