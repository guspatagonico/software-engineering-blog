# Topic: visual_effects

## Overview
Describes the MatrixBackground component that renders a multi-layer Kodama-style rain of characters with hidden gem words, mouse repulsion, and light/dark theme handling.

## Key Concepts
- Layer-based depth styling
- Gem word placement with flickering timers
- Mouse proximity forces and vortex pulls
- Performance tuning with stream counts and character-change rates
