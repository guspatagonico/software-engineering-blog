# Domain: ui

## Purpose
Capture knowledge about immersive interface visuals and animated backgrounds used across the site.

## Scope
Included in this domain:
- Canvas and WebGL based background effects
- Layered digital rain and particle simulations
- Custom UI components that respond to pointer and theme inputs

Excluded from this domain:
- Content-focused page copy, data models, and backend business logic

## Ownership
Design Systems & Frontend

## Usage
Use this domain for documenting how animated UI elements are implemented, tuned, and optimized.
